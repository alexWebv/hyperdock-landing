Features

Everything you need to talk to your dev environment.

Voice → Command

Say a task, get the exact command. Natural phrases like “open API project”, “run tests”, “build”.

Terminal, re-thought

Low-latency rendering, smart prompts and shortcuts, built to work perfectly with voice.

AI chat inline

Ask for commands, explanations, or fixes without leaving the terminal. Insert with one tap.

History & quick replay

Every command is saved. Re-run from history or pin useful sequences.

Favorites — Run & Edit

Create two types of favorites: Run (executes instantly) and Edit (opens for tweaks before run).

Git, hands-free

“Create branch feature/login”, “commit and push”, “open a PR”. Spoken workflows for Git.

Create files & folders

“New folder utils”, “make file auth/service.ts”, “append export function login”.

Themes & quick switches

Multiple themes you can toggle. Jump between projects without extra steps.
